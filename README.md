# sql-commerce-analysis
SQL E-commerce sales analysis project using MySQL joins,aggregations, and business queries.

## SQL Concepts Used
- JOINs
- GROUP BY
- Aggregate Functions
- Subqueries
- ORDER BY
- Business Analysis Queries

## Business Questions Answered
- Total Revenue
- Top Products by Revenue
- Average Order Value
- Monthly Revenue Trends
- Orders Per Customer
- Customers Without Orders
